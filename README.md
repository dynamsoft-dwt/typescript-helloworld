# DWT Demo Using TypeScript

## Prerequisites
* TypeScript

    ```bash
    npm install -g typescript
    ```

## How to Run the Demo
1. Install **dwt** package:

    ```bash
    npm install
    ```
2. Compile **test.ts**:

    ```bash
    tsc test.ts
    ```
3. Open **index.htm** in Chrome.
